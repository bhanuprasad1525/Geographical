SMART TOUR GUIDE 

Description:
------------
This web application solves a real-life problem for tourists by showing their location, current internet connectivity, and highlighting nearby attractions as they scroll.

Web APIs Used:
--------------
1. Geolocation API – to fetch and display user's current latitude and longitude.
2. Network Information API – to monitor internet connection type (4G, WiFi, etc.).
3. Intersection Observer API – to highlight landmarks when they appear in view.

How it Helps:
-------------
- Tourists can see where they are.
- Automatically get connection alerts for planning offline usage.
- Smooth experience while discovering landmarks interactively.

Tech Used:
----------
- HTML, CSS, JavaScript (Vanilla)
- Responsive layout with scroll animation
- Live feedback using browser APIs
